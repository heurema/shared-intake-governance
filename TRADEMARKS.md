# Shared Intake Governance trademark and project identity policy

The source code and documentation in this repository may be open source, but
the Shared Intake Governance name and related project identity are separate
rights.

## What the open source license covers

Unless a file says otherwise, repository code and documentation owned by the
project are licensed under the Apache License, Version 2.0.

## What the open source license does not cover

The Apache License does not grant trademark rights.

That means you may generally:

- truthfully say that your project is based on Shared Intake Governance;
- link to this repository;
- discuss Shared Intake Governance in documentation, benchmarks, and
  comparisons.

That does not mean you may:

- imply endorsement, partnership, certification, or official status;
- use the project name or identity for a fork, hosted service, package name,
  website, or company identity in a way that is likely to confuse users;
- present a modified version as the official project.

## Forks and hosted services

Forks are welcome. If you publish a fork or operate a service based on this
repository, use branding that clearly distinguishes your project from the
official repository.

## Questions

For permission or trademark questions, open a GitHub issue with the minimal
context needed to route the request.
